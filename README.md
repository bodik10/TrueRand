TrueRand
========

True Random Generator Client

Application uses random.org HTTP API for generating true randoms and showing the result in convenient way. 

Included modules: Coin, Integers, Floats, Strings, Sequences (can generate one or shuffle a custom list), Date and Time. 

App writes log. All results/logs can be exported easily. Also program uses built-in random module if random.org is unavailable.
